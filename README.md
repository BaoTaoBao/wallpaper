🇺🇸 ENGLISH - DOWNLOAD GUIDE
📥 High-Speed Sync
This tool allows you to sync any Steam Workshop item directly to your local machine using the DepotDownloader core.
Batch Download: Paste multiple IDs to download all at once.
Automatic Setup: Automatically detects and installs .NET 9.0 Runtime.
One-Click Run: Select path -> Paste ID -> Start.
🚀 Usage
Environment: Ensure Python 3.9+ is installed.
Install Library: pip install Pillow
Launch: Run python toolkit.py.
Download: Enter Workshop IDs in the "DOWNLOADER" tab and click "START SYNCING".

📥 Tải Xuống Tốc Độ Cao

🇨🇳 中文 - 下载指南
📥 高速同步功能
本工具利用 DepotDownloader 核心，支持直接从 Steam 创意工坊同步资源到本地。
批量下载: 支持一次性输入多个 ID 进行批量同步。
自动环境配置: 启动时静默安装所需的 .NET 9.0 运行库。
简化流程: 选择路径 -> 输入 ID -> 点击开始。
🚀 启动步骤
环境准备: 确认已安装 Python 3.9+。
安装依赖: 执行 pip install Pillow。
运行程序: 执行 python toolkit.py。
开始下载: 在 “创意工坊下载” 选项卡中输入 ID，点击 “开始同步数据”。
